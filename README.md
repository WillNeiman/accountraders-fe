# 채널링크 프론트엔드

이 프로젝트는 [Next.js](https://nextjs.org)를 사용하여 개발된 채널링크의 프론트엔드 애플리케이션입니다.

## 시작하기

개발 서버를 실행하려면:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 결과를 확인하세요.
